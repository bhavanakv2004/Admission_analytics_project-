# Admission_analytics_project-
Admission Funnel Analytics Dashboard using Python, Pandas, Plotly, and Streamlit to analyze student conversion rates, drop-offs, and marketing performance.
# Project Overview
This project analyzes how students move through the admission funnel and identifies key insights like conversion rates and drop-off points.
# Objectives
- Analyze student journey from lead to enrollment
- Calculate conversion rates
- Identify drop-off stages
- Evaluate marketing channel performance
- Analyze counsellor effectiveness
# Funnel Stages
Leads → Counselling → Application → Enrollment
